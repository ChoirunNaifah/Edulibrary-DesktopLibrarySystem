# 📚 EduLibrary - Sistem Perpustakaan Berbasis Desktop

Aplikasi perpustakaan berbasis desktop menggunakan Java NetBeans dan MySQL.  
Project ini dibuat sebagai tugas UKKN.

---

## 🚀 Fitur Utama

- Login Admin
- Registrasi Siswa
- CRUD Buku
- CRUD Anggota
- Peminjaman Buku
- Pengembalian Buku
- Laporan Data
